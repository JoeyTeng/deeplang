# Supported Grammer

### Primitive Type

| Key words | Type                            |
| --------- | ------------------------------- |
| `Unit`    |                                 |
| `i32`     | 32-bit integer                  |
| `i64`     | 64-bit integer                  |
| `f32`     | Single precision floating point |
| `f64`     | Double precision floating point |
| `string`  | Character string                |
| `bool`    | Logical type                    |
